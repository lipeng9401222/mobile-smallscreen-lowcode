## 🎯 核心原则

1. **版本兼容** - 所有代码必须与检测到的 Vue 版本完全兼容
2. **样式分离** - Vue 文件中禁止直接编写 CSS，统一使用外部 SCSS 文件
3. **全局变量** - `Config`、`Util`、`ejs` 等全局变量直接使用，禁止 import 导入

## 🚀 快速开始

### 1. 确定 Vue 版本

通过 `package.json` 中的 vue 依赖确定版本：

- Vue2: `"vue": "^2.x.x"`
- Vue3: `"vue": "^3.x.x"`

### 2. 选择开发模式

根据需求选择对应的开发模式：

- **列表详情页** → 参考 [列表页示例](./02-vue/examples.md#列表页面)
- **表单提交页** → 参考 [表单页示例](./02-vue/examples.md#表单页面)

## 📁 标准项目结构

```
ejs.m8.project/
├── src/
│   ├── components/          # 通用组件
│   ├── pages/               # 页面视图
│   │   └── [module_name]/   # 模块目录
│   │       ├── css/         # 样式文件
│   │       │   └── [module_name].scss
│   │       ├── images/      # 图片资源
│   │       ├── [module_name].vue
│   │       ├── router.js    # 路由配置
│   │       ├── store.js     # 状态管理
│   │       └── mock.js      # Mock 数据
│   ├── shared/              # 配置文件
│   ├── common/              # 通用文件
│   └── static/              # 静态资源
└── package.json
```

## 📋 规范检查清单

开发完成后，请确保：

- [ ] 代码与 Vue 版本兼容
- [ ] CSS 代码已分离到独立 SCSS 文件
- [ ] 全局变量（Config、Util）未使用 import 导入
- [ ] 所有输入框已设置 placeholder
- [ ] readonly 输入框已添加 `is-link` 和 `clickable` 属性
- [ ] 组件 props、data、methods 已添加中文注释
- [ ] 文件头部已添加标准注释
- [ ] 路由已在 router.js 中正确配置
