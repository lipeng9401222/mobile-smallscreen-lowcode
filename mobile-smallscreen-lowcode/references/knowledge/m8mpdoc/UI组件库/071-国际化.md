# 国际化

### 介绍

-   跨端组件库采用中文作为默认语言，同时支持多语言切换，请按照下方教程进行国际化设置。

### 使用方法

#### 多语言切换

组件库通过 `Locale` 组件实现多语言支持，使用 `Locale.use` 方法可以切换当前使用的语言。

```js
// 由于 Locale 是全局函数，可以不用手动引入该函数
import Locale from '@components/locale/index.js';
// 引入英文语言包
import enUS from '@components/locale/i18n/en.json';

Locale.use('en', enUS);
```

#### 覆盖语言包

通过 `Locale.add` 方法可以实现文案的修改和扩展，示例如下：

```js
const messages = {
    'zh-Hans': {
        emPicker: {
            confirm: '关闭' // 将'确认'修改为'关闭'
        }
    }
};

Locale.add(messages);
```

#### 语言包

目前支持的语言:

| 语言     |   key   | 文件名       |
| :------- | :-----: | :----------- |
| 简体中文 | zh-Hans | zh-Hans.json |
| 英文     |   en    | en.json      |

在[这里](待补充)查看所有语言文件

若需要翻译成其他语言，可以使用`Locale.add`进行扩展。

业务代码请使用 [vue-i18n](https://github.com/kazupon/vue-i18n) 实现。
