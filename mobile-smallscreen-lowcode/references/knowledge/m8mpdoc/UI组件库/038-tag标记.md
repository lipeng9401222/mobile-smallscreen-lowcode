# 组件使用

## tag 标记

### 介绍

-   标记组件。

::: ifdef M83

-   `v8.3.11`开始支持 weex 端使用

::: endif

### 代码演示

#### 基础用法

通过 `type` 属性控制标签颜色。

```html
<em-tag type="primary">标签</em-tag>
<em-tag type="success">标签</em-tag>
<em-tag type="danger">标签</em-tag>
<em-tag type="warning">标签</em-tag>
```

#### 基座风格: 人员、部门、职级等

通过`basic-type`属性设置为`person`可使用基座风格的人员标签, 该类型下通过`photo`属性可设置头像

```html
<em-tag basic-type="person">科长</em-tag>
<em-tag basic-type="person" closeable @close="close()">我的部门</em-tag>
<em-tag basic-type="person" photo="https://img01.yzcdn.cn/vant/cat.jpeg">李科员</em-tag>
```

#### 基座风格: 属性标签

通过`basic-type`属性设置为`type`可使用基座风格的属性标签, 基座风格的标签背景色都符合基座的规范

```html
<em-tag type="warning" basic-type="type">待审核</em-tag>
<em-tag type="danger" basic-type="type">特急件</em-tag>
<em-tag type="primary" basic-type="type">待分送</em-tag>
```

#### 基座风格: 新闻标题标签

通过`basic-type`属性设置为`news`可使用基座风格的新闻标题标签, 基座风格的标签背景色都符合基座的规范

```html
<em-tag type="danger" basic-type="news">急</em-tag>
<em-tag type="primary" basic-type="news">政策</em-tag>
<em-tag type="danger" basic-type="news">热点</em-tag>
```

#### 基座风格: 状态标签

通过`basic-type`属性设置为`status`可使用基座风格的状态标签, 该类型下通过`status`属性可设置状态, 只支持示例上的几种状态

```html
<em-tag basic-type="status" status="complete"></em-tag>
<em-tag basic-type="status" status="loading"></em-tag>
<em-tag basic-type="status" status="offline"></em-tag>
```

#### 基座风格: 数值标签

通过`basic-type`属性设置为`number`可使用基座风格的状态标签, 基座风格的标签背景色都符合基座的规范

```html
<em-tag basic-type="number">12</em-tag>
```

#### 圆角样式

通过 `round` 设置为圆角样式。

```html
<em-tag round type="primary">标签</em-tag>
<em-tag round type="success">标签</em-tag>
<em-tag round type="danger">标签</em-tag>
<em-tag round type="warning">标签</van-tag>
```

#### 标记样式

通过 `mark` 设置为标记样式(半圆角)。

```html
<em-tag mark type="primary">标签</em-tag>
<em-tag mark type="success">标签</em-tag>
<em-tag mark type="danger">标签</em-tag>
<em-tag mark type="warning">标签</em-tag>
```

#### 空心样式

设置 `plain` 属性设置为空心样式。

```html
<em-tag plain type="primary">标签</em-tag>
<em-tag plain type="success">标签</em-tag>
<em-tag plain type="danger">标签</em-tag>
<em-tag plain type="warning">标签</em-tag>
```

#### 自定义颜色

```html
<em-tag color="#f2826a">标签</em-tag>
<em-tag color="#7232dd">标签</em-tag>
<em-tag color="#7232dd" plain>标签</em-tag>
<em-tag color="#ffe1e1" text-color="#ad0000">标签</em-tag>
```

#### 标签大小

```html
<em-tag type="danger">标签</em-tag>
<em-tag type="danger" size="medium">标签</em-tag>
<em-tag type="danger" size="large">标签</em-tag>
```

#### 可关闭标签

添加 `closeable` 属性表示标签是可关闭的，关闭标签时会触发 `close` 事件，在 `close` 事件中可以执行隐藏标签的逻辑。

```html
<em-tag v-if="show.primary" closeable size="medium" type="primary" @close="close('primary')">标签</em-tag>
<em-tag v-if="show.success" closeable size="medium" type="success" @close="close('success')">标签</em-tag>
```

::: ifdef M83

```js
export default {
    data() {
        return {
            show: {
                primary: true,
                success: true
            }
        };
    },
    methods: {
        close(type) {
            this.show[type] = false;
        }
    }
};
```

::: endif
::: ifdef M84

```js
import { ref } from 'vue';
const show = ref({
    primary: true,
    success: true
});

const close = (type) => {
    show.value[type] = false;
};
```

::: endif

### API

#### Props

| 参数       |                                      说明                                      | 类型    | 默认值    | ::: ifdef M83 weex 端支持 ::: endif              |
| :--------- | :----------------------------------------------------------------------------: | :------ | :-------- | :----------------------------------------------- |
| type       |              类型，可选值为`primary` `success` `danger` `warning`              | string  | `default` | 是                                               |
| size       |                         大小, 可选值为`large` `medium`                         | string  | -         | 是                                               |
| color      |                                    标签颜色                                    | string  | -         | 是                                               |
| plain      |                                 是否为空心样式                                 | boolean | `false`   | 是                                               |
| round      |                                 是否为圆角样式                                 | boolean | `false`   | 是                                               |
| mark       |                                 是否为标记样式                                 | boolean | `false`   | 是                                               |
| text-color |                        文本颜色，优先级高于`color`属性                         | string  | `white`   | 是                                               |
| closeable  |                                是否为可关闭标签                                | boolean | `false`   | 是                                               |
| basic-type |       基座风格的标签类型, 可选值为` person``type``news``status``number `       | string  | `-`       | 是（在 weex 端基座类型样式都是固定的，无法修改） |
| photo      |                     基座人员类型标签时，可用该属性设置头像                     | string  | `-`       | 是                                               |
| status     | 基座状态类型标签时，可用该属性设置状态，可选值为` complete``loading``offline ` | string  | `-`       | 是                                               |

#### Slots

| 名称    |     说明     |
| :------ | :----------: |
| default | 标签显示内容 |

#### Events

| 事件名 |      说明      | 回调参数     |
| :----- | :------------: | :----------- |
| click  |   点击时触发   | event: Event |
| close  | 关闭标签时触发 | -            |
