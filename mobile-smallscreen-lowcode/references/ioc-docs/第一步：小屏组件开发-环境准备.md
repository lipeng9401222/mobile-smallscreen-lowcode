# 第一步：环境准备（小屏组件开发）



## 1. Node环境要求
###  版本要求

**推荐版本：** Node.js v16.20.0 （建议使用 LTS 长期支持版本）
#### 方法一：官网下载安装（推荐新手）

1. **访问官网**：https://nodejs.org/

2. **选择版本**：

   - LTS（长期支持版）：稳定可靠，推荐生产环境使用
   - Current（当前版本）：包含最新特性

3. **下载安装包**：

   - Windows：下载 `.msi` 安装包
   - macOS：下载 `.pkg` 安装包
   - Linux：下载对应发行版的安装包

4. **运行安装程序**：按照提示完成安装

5. **验证安装**：

   ```bash
   node -v
   npm -v
   ```

#### 方法二：使用 nvm 管理多版本（推荐开发者）

**nvm（Node Version Manager）** 可以在同一台机器上安装和切换多个 Node.js 版本。

**Windows 安装 nvm：**

1. 下载 nvm-windows：https://github.com/coreybutler/nvm-windows/releases

2. 下载 `nvm-setup.exe` 并安装

3. 打开命令行，验证安装：

   ```bash
   nvm version
   ```

**macOS/Linux 安装 nvm：**

```bash
# 使用 curl 安装
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 或使用 wget 安装
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

**使用 nvm 安装 Node.js：**

```bash
# 查看可用版本
nvm list available

# 安装指定版本
nvm install 16.20.0

# 使用指定版本
nvm use 16.20.0

# 设置默认版本
nvm alias default 16.20.0
```
