# IOC 参考模板索引

只打开与当前任务最接近的 1 到 2 个模板，不要一次性把全部模板内容加载到上下文里。

## 模板选择建议

### `MineEvaluateAdd`

适合：

- 售后申请
- 评价表单
- 文本说明
- 上传图片或视频
- 提交按钮

关注点：

- `em-rate`
- `textarea`
- `em-uploader`
- `onSubmit`

### `OrderPay`

适合：

- 订单确认
- 金额小计
- 底部提交栏
- 备注输入

关注点：

- 金额展示
- 列表小计
- 底部操作条

### `ShoppingCart`

适合：

- 商品列表
- 数量增减
- 勾选
- 汇总

关注点：

- `em-stepper`
- `em-checkbox`
- 列表事件

### `GoodsInfo`

适合：

- 商品标题
- 价格信息
- 简要说明区

关注点：

- 标题排版
- 价格拆分
- 标签信息

### `GoodsSimilar`

适合：

- 滑动卡片
- 轮播展示
- 图片宫格

关注点：

- `em-swipe`
- 图片卡片布局
