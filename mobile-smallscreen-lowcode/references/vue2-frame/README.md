# 框架简介

## M8移动前端跨端框架

`M8移动前端跨端框架`是使用`Vue.js`开发跨平台移动前端应用的框架，开发人员编写一套代码，可发布到`H5`、`EJS`、`微信小程序`、`支付宝小程序`多个平台。

基于公司`M8移动前端框架`迭代升级而来，保留了一致的开发体验，提供了跨端构建能力。

对`M8移动前端框架`的UI组件库，Util工具库，EJSAPI均进行了跨平台改造支持。

通过使用框架，可以快速的进行移动跨端类项目开发。

## 为什么要有`M8`框架

现如今市面上端的形态多种多样，H5、APP、小程序等各种端大行其道。

当业务要求同时在不同的端都要求有所表现的时候，针对不同的端去编写多套代码的成本显然非常高，这时候只编写一套代码就能够适配到多端的能力就显得极为需要。

同时移动前端框架也收到产品项目对多端构建的迫切需求，需要一套代码运行在APP、H5与各端小程序的能力。

随着市面上端的多样化，针对同一个应用需要在不同平台发布的需求，迫切需要一个只维护一套代码就可以构建多入口的解决方案。

让开发人员只需要关注业务层，不需要关注底层代码的不同实现，节约开发成本。

## `M8`框架能做什么

框架适用于移动跨平台项目的开发，且不仅限于`H5端`

如下情况都可采用 `M8移动前端跨端框架`

- H5端项目开发

- 新点EJS-Hybrid项目

- 新点小程序项目

- 微信小程序项目

- 支付宝小程序项目
